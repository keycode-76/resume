Pocony
A little balcony on your phone
スマホの中の小さなベランダ
Flutter Indie Game Mobile
What it is
概要

Pocony is a mobile idle farming game built with Flutter. Plants grow in real time based on actual weather data pulled from your location, sunny days speed up growth, rainy days water your garden automatically. Leave your phone for a week, come back to a completely different balcony.

Poconyは、Flutterで開発した放置型ファームゲームです。現地の天気データをリアルタイムで取得し、晴れた日は成長が加速し、雨の日は自動で水やり。1週間放置して戻ると、全く違う景色のベランダが待っています。

Key Features
主な機能
  • Real time plant growth driven by live weather data
  • Interactive planting system (till, plant, water, grow, harvest)
  • Home screen farm widget integration for quick status access
  • Persistent local game state with lightweight on device storage
  • リアルタイム天気データに基づく植物の成長システム
  • インタラクティブな栽培フロー（耕す・植える・水やり・成長・収穫）
  • ホーム画面ウィジェットでゲーム状況をすぐに確認
  • 軽量ローカルストレージによるゲーム状態の永続化
Tech Stack

  • Isar Community
  • Open-Meteo API

担当

100% solo, product concept, game design, UI/UX, Flutter development, weather API integration, game loop logic, and app store submission.

課題

The hardest part: The most challenging part is the game design, calculating plant growth speed based on weather, and building the game loop for flowering, weeding, and planting.

AI Agent Web Game Hackathon
What it is
概要

A social deduction game where 1 human joins a group chat with 4 AI agents, and nobody knows who's real. Players take turns speaking, then vote to eliminate who they think is AI. Built in 48 hours at Tokyo Hackathon 2026, winning 2nd place.

1人の人間と4体のAIエージェントが同じグループチャットに参加し、誰がAIかを投票で見破るゲームです。東京ハッカソン2026で48時間制作し、2位受賞。

Key Features
主な機能
  • Turn-based game loop: speak → vote → eliminate → next round
  • AI agents with trust/evasiveness impression scoring (Mistral API)
  • Real-time multiplayer state sync via WebSocket
  • Browser Web Speech API for AI text-to-speech
  • DiceBear SVG avatars with smooth position interpolation
  • ターン制ゲームループ：発言 → 投票 → 脱落 → 次のラウンド
  • Mistral APIによるAIエージェントの信頼度・回避度スコアリング
  • WebSocketによるリアルタイム多人数同期
  • ブラウザ Web Speech APIを使ったAIのテキスト読み上げ
  • DiceBear SVGアバターとスムーズな位置補間アニメーション
Tech Stack
技術スタック
  • Next.js 14 + TypeScript + Tailwind CSS
  • Express.js + WebSocket
  • Mistral API
  • Web Speech API
  • Supabase (design-layer)
  • Next.js 14 + TypeScript + Tailwind CSS
  • Express.js + WebSocket

  • Web Speech API

担当

Game design, UI/UX design, full frontend development.

ゲームデザイン・UI/UX設計・フロントエンド全般・AIエージェントのプロンプトエンジニアリング・リアルタイムセッション管理を担当。

Challog
A challenge tracker to build your streaks
ストリークを作るチャレンジトラッカー
React Native Google Play Productivity
What it is
概要

Challog is a minimalist daily challenge tracker built to help users stay consistent. Create challenges, log daily check ins, watch your streak grow, and share progress snapshots, with motivational nudges to keep the momentum going.

Challogは、習慣継続を支援するシンプルなチャレンジ記録アプリです。毎日チェックインし、ストリークを伸ばし、進捗スナップショットを共有できます。

Key Features
主な機能
  • Daily challenge check-in with streak tracking and status display
  • Progress snapshot generation and one-tap sharing
  • Google Sign In with Supabase cloud sync and local/cloud merge logic
  • Android home screen widget syncing challenge data to the native layer
  • In-app purchases via RevenueCat for Premium upgrade
  • Full multilingual support with i18next
  • 毎日のチェックインとストリーク追跡・ステータス表示
  • 進捗スナップショットの生成とワンタップ共有

  • RevenueCatによるアプリ内課金・プレミアムアップグレード
